# Proyecto final – Bases de Datos Distribuidas ENOE/ENOEN

> Diseño centralizado, fragmentación horizontal y migración de SQLite a PostgreSQL.

---

## Resumen
Este proyecto desarrolla una base de datos centralizada a partir de la ENOE/ENOEN, normalizada en tercera forma normal, y posteriormente implementa un esquema distribuido por periodos.  
El entorno original de trabajo fue **SQLite**, debido a restricciones de instalación en la computadora utilizada. Para facilitar la revisión, se realizó además una migración complementaria a **PostgreSQL** y se generaron respaldos equivalentes.
